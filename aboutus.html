Scrroll-In

"You can save the scroll position of the page and then revisit the page at any time to continue where you left. Also, extension allows you to save multiple scrolls from the page. You can fetch the last saved scroll. If you don't want a scroll you can delete it. Extension allows you to see all the saved scrolls arranged in the newly first order where you can delete any specific scroll or clear all the Scrolls at once. You can use update the scroll which will update the last save scroll with the current one. You can add customisable keyboard shortcuts to save, fetch or delete scrolls without having to open the extension popup. The extension auto fetches the last saved scroll position on the page load, if page has saved any scroll.

Completely free and open-source:
Scrroll In was and will always be free to use and open-source.
Feel free to open an issue for feature request or bug report on issues pages on GitHub.
